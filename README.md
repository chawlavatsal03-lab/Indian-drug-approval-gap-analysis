## ?? Executive Summary 
 
This project analyzes alignment between CDSCO drug approvals and non-communicable disease burden in India using WHO GBD 2019 data. 
 
**Key Finding:** System shows +1.31 average gap. Inflammation and Immunology (+6 each) most under-served. Cardiovascular (-8) severely over-served.
